Горячие клавиши:
Ctrl + c - копировать
Ctrl + v - вставить
Ctrl + x - вырезать
Ctrl + s - сохранить
Ctrl + z - отмена
Ctrl + Shift + z - вперед (отменить отмену)
Ctrl + a - выделить все
Ctrl + n - новый файл
Ctrl + o - открыть
Ctrl + d - копировать строку
Ctrl + p - печать
Ctrl + Shift + n - создать новую папку
Ctrl + крутим колесико - приблизить/отдалить
или
Ctrl + -
Ctrl + + - тоже приблизить/отдалить

Alt + F4 - закрыть программу
Alt + Tab - переключиться между программами
Alt + Enter - открыть игру на все окно

Esc - выход

Нажатие на колесеко - закрыть вкладку браузера или программы или открыть сайт в новой вкладке

Ctrl + Win + Alt + L - выравнивание кода в IDEA
